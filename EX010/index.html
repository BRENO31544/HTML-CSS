Links nofollow

A ideia desse atributo é voltada ao rankeamento em mecanismos de busca. A o atribuir o "rel="nofollow"" a um link estamos dizendo a um site - "não olhe para o rankeamento dese link". Isso pq o rankeamento do link citado pode não ser benéfico para o meu site ainda que seja é possível que o site citado não queira estar atrelado em sites terceiros.

Links dofollow

Sendo o contrário do anterior, esse usa o rankeamento atrelado. Importante citar que esse é incluso automaticamente a tag (ainda que não esteja escrito em linhas código) quando não há o atributo nofollow

Mapa do metro

Ainda com esse atributo, dependendo do navegador é possível que o arquivo não seja necessáriamente baixado e nesse caso temos como incluir no arquivo o atributo type. Esse por sua vez serve para "forçar" o link a agir da forma necessidade do usuário uma vez que com a tag type colocada é necessario literalemnte inserir o tipo de arquivo do qual o link se trata. Daí basta inserir um tipo de arquivo específico para PDF; JPG; PNG; XLSX ... e daí por diante. uma forma de descobrir quais são essas "extensões" para cada tipo de arquivo temos o site iana.org com as mais diversas extensões para inserir nessa tag

* Uma forma de encontrar informações nessa pagina mais rápido é usando as teclas ctrl + F e digitanto a media type desejada

Uma vez com a type do arquivo devidamente colocada, conseguimos baixar o arquivo
